\subsection{Observability Stack}
\begin{itemize}
  \item MLflow: experiment tracker, model registry, and inter-flow contract
  \item DuckDB: in-process SQL analytics over \texttt{mlruns/} (no server required)
  \item Langfuse (self-hosted): LLM tracing and cost tracking for agentic dev story
  \item Prometheus + Grafana: infrastructure metrics (GPU/CPU/memory/disk)
  \item Evidently: data and model drift detection with Grafana export
  \item OpenLineage (Marquez): IEC~62304-compliant data lineage tracking
  \item Label Studio: multi-annotator workflows for ground truth quality
  \item Cleanlab: label quality assessment and noisy label detection
  \item whylogs: lightweight data profiling at pipeline entry
\end{itemize}
