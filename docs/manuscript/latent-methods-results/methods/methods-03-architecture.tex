\subsection{Platform Architecture}
\begin{itemize}
  \item 5-flow Prefect DAG with Docker-per-flow isolation (6 containers total)
  \item MLflow is the ONLY inter-flow contract --- no shared filesystem between flows
  \item Layer 0 (Infrastructure): PostgreSQL, MinIO, MLflow, Prefect Server, Grafana
  \item Layer 1 (GPU Management): NVIDIA MIG for multi-model inference partitioning
  \item Layer 2 (Training Execution): Optuna ASHA + PyTorch DDP
  \item Layer 3 (Compute Provisioning): SkyPilot multi-cloud spot instances
  \item Layer 4 (Orchestration): Prefect 3.x Server with Docker workers
\end{itemize}
