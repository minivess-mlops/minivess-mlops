\subsection{Overview}
\begin{itemize}
  \item ARBOR: open-source MLOps platform for reproducible 3D biomedical segmentation
  \item Platform-as-contribution: enables SOTA, does not claim SOTA
  \item MONAI ecosystem extension --- any MONAI model, any 3D segmentation dataset
  \item MIT license --- reproducible, modifiable, deployable
  \item Addresses reproducibility crisis in computational biology (cite: Wittmann 2024, Poon 2023)
\end{itemize}
