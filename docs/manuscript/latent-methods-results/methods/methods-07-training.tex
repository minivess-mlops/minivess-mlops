\subsection{Training Pipeline}
\begin{itemize}
  \item Default loss: \texttt{cbdice\_cldice} (CbDiceClDiceLoss) --- topology-optimized for vasculature
  \item Framework: vanilla PyTorch + MONAI (no PyTorch Lightning --- autolog incompatible)
  \item AMP policy: ON for training, OFF for validation
        (MONAI issue \#4243 --- 3D operations + autocast $\to$ NaN in validation)
  \item Val-skip sentinel: \texttt{val\_interval > max\_epochs} disables validation
        (OOM guard for large models, e.g.\ SAM3 Hybrid on 8~GB GPU)
  \item HPO: Optuna with ASHA (HyperbandPruner), TPE/CmaES samplers
  \item Checkpoint files: \texttt{best\_val\_loss.pth}, \texttt{last.pth}, \texttt{epoch\_latest.pth}
  \item Reproducibility: seed=42, SHA-256 checksums on all artifacts
\end{itemize}
