\subsection{Agentic Development Methodology}
\begin{itemize}
  \item Claude Code + \texttt{CLAUDE.md} as living software design document (SDD-inspired)
  \item 52-node probabilistic knowledge graph as Bayesian decision network
  \item TDD-first workflow: self-learning-iterative-coder Skill
        (red $\to$ green $\to$ verify $\to$ fix $\to$ checkpoint $\to$ converge)
  \item Metalearning: \texttt{.claude/metalearning/} --- failure analyses persisted to files,
        never discarded to terminal output
  \item Intent expression: \texttt{CLAUDE.md} ban list + decision node \texttt{intent\_sources}
  \item Phase 1 belief propagation: \texttt{requires\_review} flags trigger on spec change
  \item Phase 2 (future): actual posterior updates when experimental results arrive
\end{itemize}
