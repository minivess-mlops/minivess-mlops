\subsection{Workflow Orchestration}
\begin{itemize}
  \item Prefect 3.x REQUIRED --- not optional scaffolding, enforced at runtime
  \item Five flows mapping to researcher personas:
    \begin{enumerate}
      \item Data Engineering (Data Engineer persona) --- core, stops pipeline on failure
      \item Model Training (ML Engineer persona) --- core, stops pipeline on failure
      \item Model Analysis (Statistician persona) --- core, stops pipeline on failure
      \item Deployment (MLOps persona) --- core, stops pipeline on failure
      \item Dashboard \& Reporting (Researcher persona) --- best-effort, failure non-blocking
    \end{enumerate}
  \item \texttt{PipelineTriggerChain} handles cascading execution and spot recovery
  \item Inter-flow contract: MLflow run tags (\texttt{flow\_name='training-flow'})
  \item Flows communicate ONLY through MLflow artifacts --- no shared filesystem
\end{itemize}
