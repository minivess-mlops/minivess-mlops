\subsection{Deployment and Serving}
\begin{itemize}
  \item ONNX export via BentoML + ONNX Runtime (CPU and GPU inference paths)
  \item Champion model tagging in MLflow registry (Flow 4 --- Deployment flow)
  \item Gradio demo interface for interactive segmentation visualization
  \item NVIDIA MIG partitioning for multi-model concurrent inference on single GPU
  \item BentoML service definitions in \texttt{src/minivess/serving/}
  \item Model registry follows ONNX-first policy --- \texttt{.onnx} files gitignored
        (stored in MLflow/BentoML registries, not git)
\end{itemize}
